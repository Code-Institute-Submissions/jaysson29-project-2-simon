About Simon Game :

Simon is an electronic game of memory skill invented by Ralph H. Baer and Howard J. Morrison, working for toy design firm Marvin Glass and Associates, with software programming by Lenny Cope. The device creates a series of tones and lights and requires a user to repeat the sequence ( Find Out More Here ! ). I Have Created A JavaScipt Version Which Uses The Same Algorithm. I Used Colored Planets Instead Of Just Color To Make It Unique.

How To Play

  • Click Play On The HomePage To Take You To The Game.
  • Click Start On The Game Board To Begin Playing.
  • If You Want To Play Strict Mode (Read More In About Section) Use The Toggle To Enable or Disable Strict Mode.
  • When The Game Starts It Will Randomly Select A Colored Planet ( Green Red Yellow Blue ) As The Starting Point.
  • When A Color Is Selected, It will Flash With A Brighter Version Of The Color With A Outline And Will Make A Unique Sound To define Each Color.
  • Use Your Mouse Pointer To Select The Color You Seen To Go To The Next Round.
  • Every Round Will Increases The Amount Of Color's In The Sequence, For Example; , , , ... And The Score Will Increase As Well By 1.
  • Make Sure After Every Selection You Center The Cursor So The Sequence Can Be Seen Clearly, If The Cursor Is Hovering Over A Color/Planet It Will Override The Sequence And You Will Not Be Able To Continue.
  • Make Sequence Number Is Currently Set At 20.
  • Finally Have Fun.

    • Try And Get The Highest Score, If Your Dare
